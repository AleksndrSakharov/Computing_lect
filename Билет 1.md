```
Понятие корректности задачи по Адамару. Три условия корректности (существование, единственность, устойчивость). Примеры корректных (интегрирование) и некорректных (дифференцирование) задач.
```

#Определение Горорят, что задача *корректна*, с точки зрения вычислительной математики, если:
1. $\exists$ решение задачи
2. Решение единственное
3. Малому изменению начальных данных соответствует малое изменение выходных данных

#Пример 
### Задача численного интегрирования

Требуется вычислить $\int_{a}^{b} f(x) dx = I$, где $f(x)$ — непрерывная функция. Интеграл по Риману существует.

Пусть $\delta(x)$ — непрерывная функция.

#### Предположим
Что на вход дана функция $\tilde{f}(x) = f(x) + \delta(x)$, где $||\delta(x)|| = \delta$ — малое число: $max|\delta(x) - 0| = \delta$ 
$|\tilde{I} - I| = |\int_{a}^{b} \delta(x) dx| \leq \int_{a}^{b} |\delta(x) dx| \leq |\delta\int_{a}^{b} dx| = \delta(b - a) \Rightarrow$ Выполняется *3* из определения корректной задачи с точки зрения вычислительной математики $\Rightarrow$ **Интегрирование — корректная задача**.

#Пример
### Задача численного дифференцирования

Рассмотрим вариант получения производной и докажем, что дифференцирование — некорректная задача.

Пусть  
$f(x) = \dfrac{1}{N} \sin(N^2 x)$.

При увеличении $N$ уменьшается $|f(x)|$.

Производная:  
$f'(x) = \dfrac{N^2}{N} \cos(N^2 x) = N \cos(N^2 x)$.

Для удобства производную берём на $[a, b]$.

$f'(x)$ — периодическая, где  $T = \dfrac{2\pi}{N^2}$.

С некоторого номера  $\dfrac{2\pi}{N^2} < b - a$  $\Rightarrow \exists x_0 \in [a, b] : \cos(N^2 x_0) = 1 \Rightarrow |f'(x_0)| = N$.


Рассмотрим $F(x)$ и $F(x) + f(x)$, где $c \uparrow N$, происходит $\downarrow |f(x)|$, то есть имеем малое изменение начальных данных.

$∣(F(x_0​)−f(x_0​))'−F′(x_0​)∣=∣f′(x_0​)∣=N$,

получили ранее для $x = x_0$.

НО имеем увеличивающееся с $\uparrow N$ изменение выходных данных.

$\Rightarrow$ **не выполняется (3)** из определения корректной задачи с точки зрения вычислительной математики.

$\Rightarrow$ **Дифференцирование — некорректная задача.**
