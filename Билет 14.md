**1. Метод простой итерации**
Для решения $Ax=b$ систему преобразуют к виду $x = Bx + c$.
Алгоритм: $x^{(k+1)} = B x^{(k)} + c$.
Условие сходимости: $||B|| < 1$ (любая каноническая норма матрицы).
Достаточное условие сходимости для исходной матрицы $A$ — **диагональное преобладание**: $|a_{ii}| > \sum_{j \ne i} |a_{ij}|$.

**2. Число обусловленности**
Характеризует зависимость погрешности решения от погрешности входных данных.
$$\mu(A) = ||A|| \cdot ||A^{-1}||$$
Оценка относительной погрешности:
$$\frac{||\Delta x||}{||x||} \le \mu(A) \frac{||\Delta b||}{||b||}$$
Если $\mu(A)$ велико — система *плохо обусловлена* (малая ошибка в данных дает огромную ошибку в ответе). Оптимально $\mu(A) = 1$ (для ортогональных матриц).

**3. Метод Зейделя**
Модификация метода простой итерации. При вычислении $(k+1)$-го приближения для переменной $x_i$ используются уже найденные на этом же шаге значения $x_1^{(k+1)}, \dots, x_{i-1}^{(k+1)}$.
Формально: матрица $A$ разбивается на $L$ (нижнюю), $D$ (диагональ), $U$ (верхнюю).
Итерация: $(L+D)x^{(k+1)} = b - U x^{(k)}$.
Обычно сходится быстрее метода простой итерации.

## 1. Метод простой итерации.

Дана СЛАУ $Ax = b$. Преобразуем к виду $x = Bx + c$, где $B = E - Q^{-1}A$, $c = Q^{-1}b$, $Q$ — невырожденная матрица (часто $Q = \omega^{-1}D$ или $Q = D$ для метода Якоби).

Итерационный процесс:
$$
x^{(k+1)} = Bx^{(k)} + c, \quad k = 0, 1, 2, \ldots
$$

## 2. Метод Зейделя.
Разложим $A = L + D + U$, где $D$ — диагональная, $L$ — нижняя треугольная, $U$ — верхняя треугольная матрицы.

Метод Зейделя:
$$
(D + L)x^{(k+1)} = b - Ux^{(k)},
$$

или в покомпонентной форме:
$$
x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j=1}^{i-1} a_{ij}x_j^{(k+1)} - \sum_{j=i+1}^n a_{ij}x_j^{(k)} \right), \quad i = 1, \ldots, n.
$$

## 3. Условия сходимости.

Из теоремы Банаха ([[Билет 13]]) метод сходится, если отображение $\Phi(x) = Bx + c$ является сжимающим в некоторой норме, т.е. $||B|| < 1$.

**Достаточные условия сходимости:**

- **Условие по норме матрицы:** Если $||B|| < 1$ в какой-либо индуцированной норме (например, $||B||_{\infty}, ||B||_1, ||B||_2$), то метод простой итерации сходится.
- **Условие диагонального преобладания:**
  - Строгое диагональное преобладание по строкам:
$$
|a_{ii}| > \sum_{j\neq i} |a_{ij}|, \quad i = 1, \ldots, n.
$$
	Тогда метод Якоби и метод Зейделя сходятся.
  - Строгое диагональное преобладание по столбцам:
$$
|a_{jj}| > \sum_{i\neq j} |a_{ij}|, \quad j = 1, \ldots, n.
$$
	Тогда метод Якоби и метод Зейделя сходятся.

## 4. Доказательство сходимости при диагональном преобладании (для метода Якоби).

Для метода Якоби: $B = -D^{-1}(L + U)$. Покажем, что $\|B\|_{\infty} < 1$.
В норме $\|\cdot\|_{\infty}$:
$$
\|B\|_{\infty} = \max_i \sum_{j=1}^n |b_{ij}| = \max_i \frac{1}{|a_{ii}|} \sum_{j\neq i} |a_{ij}|.
$$
Из условия строгого диагонального преобладания:
$$
\frac{1}{|a_{ii}|} \sum_{j\neq i} |a_{ij}| < 1 \quad \text{для всех } i.
$$
Следовательно, $\|B\|_{\infty} < 1$ → метод Якоби сходится.
