## #Определение Нормы вектора (аксиомы):

Пусть $X$ — векторное пространство. **Нормой** элемента $x \in X$ называется неотрицательное число $\|x\|$, удовлетворяющее условиям:
1. **Аксиома отдельности:**  $\|x\| = 0 \iff x = 0;$
2. **Аксиома абсолютной однородности:**  $\|\alpha x\| = |\alpha| \cdot \|x\|, \quad \forall \alpha \in \mathbb{R} (\text{или } \mathbb{C}), \forall x \in X;$
3. **Аксиома субаддитивности (неравенство треугольника):**  $\|x + y\| \leq \|x\| + \|y\|, \quad \forall x, y \in X.$

**Связь между нормой и метрикой:** В нормированном пространстве можно канонически ввести метрику:  $\rho(x, y) = \|x - y\|.$

**Примеры норм:**
- В $\mathbb{R}^n$:  $\|x\|_2 = \sqrt{\sum |x_i|^2}, \quad \|x\|_1 = \sum |x_i|, \quad \|x\|_\infty = \max |x_i|.$
- В $C[a, b]$:  $\|f\| = \max_{t \in [a, b]} |f(t)|.$

## #Доказательство, что $\rho(x,y) = \|x - y\|$ является метрикой:

1. $\rho(x,y) = \|x - y\| \geq 0$ по определению нормы.  $\rho(x,y) = 0 \Leftrightarrow \|x - y\| = 0 \Leftrightarrow x - y = 0 \Leftrightarrow x = y.$
2. $\rho(x,y) = \|x - y\| = \|(-1)(y - x)\| = |-1| \cdot \|y - x\| = \|y - x\| = \rho(y,x).$
3. $\rho(x,z) = \|x - z\| = \|(x - y) + (y - z)\| \leq \|x - y\| + \|y - z\| = \rho(x,y) + \rho(y,z).$
