**1. Остаточный член $R(f)$**
Погрешность выводится через разложение в ряд Тейлора или через ядро Пеано.
*   Для формулы **средних прямоугольников**: $R = \frac{(b-a)^3}{24} f''(\xi)$.
*   Для формулы **трапеций**: $R = -\frac{(b-a)^3}{12} f''(\xi)$.
*   Для формулы **Симпсона**: $R = -\frac{(b-a)^5}{2880} f^{(4)}(\xi)$ (очень высокая точность для гладких функций).

Для составных формул (весь отрезок разбит на $N$ шагов длины $h$) порядок погрешности снижается на единицу степени длины отрезка, но выражается через шаг $h$:
Трапеции: $O(h^2)$. Симпсон: $O(h^4)$.

## Вывод остаточных членов

Остаточный член $E[f]$ квадратурной формулы $I[f] = (b - a) \sum_{k=0}^s w_k f(x_k)$ находится с помощью формулы Тейлора с интегральным остатком.

**Общий вид остаточного члена:**
$$
E[f] = \int_a^b \frac{f^{(m+1)}(t)}{m!} Q(t) dt,
$$
где  $Q(t) = \int_t^b (x - t)^m dx - (b - a) \sum_{k: x_k > t} w_k (x_k - t)^m$ — функция, зависящая от конкретной формулы степени $m$.

**Конкретные формулы (для стандартного отрезка):**

- **Формула средней точки ($m = 1$):** $E[f] = \frac{f''(\zeta)(b - a)^3}{24}$
- **Формула трапеций ($m = 1$):** $E[f] = \frac{f''(\zeta)(b - a)^3}{12}$
- **Формула Симпсона ($m = 3$):** $E[f] = \frac{f^{(4)}(\zeta)(b - a)^5}{2880}$

## Вывод остаточного члена для формулы средней точки

1. Для формулы средней точки ($m = 1$) имеем один узел $x_0 = \frac{a+b}{2}$, вес $w_0 = 1$.
2. Функция $Q(t)$ для степени $m = 1$:
$$
Q(t) = \int_t^b (x - t) dx - (b - a) \cdot 1 \cdot \left( \frac{a + b}{2} - t \right)_+,
$$
где $(z)_+ = \max(z, 0)$.

3. После вычисления интеграла и учёта того, что $\frac{a+b}{2}$ лежит внутри $[a, b]$, получаем:
$$
Q(t) = \frac{(b - t)^2}{2} - (b - a) \left( \frac{a + b}{2} - t \right) \quad \text{для } t \leq \frac{a + b}{2}.
$$
4. По теореме об интегральном среднем:
$$
E[f] = \int_a^b \frac{f''(t)}{1!} Q(t) dt = f''(\zeta) \int_a^b Q(t) dt, \quad \zeta \in [a, b].
$$
5. Вычисляя $\int_a^b Q(t) dt$, получаем $\frac{(b-a)^3}{24}$.

6. Итог: $E[f] = \frac{f''(\zeta)(b-a)^3}{24}$.
